# Mobilna aplikacja edukacyjna „Mini Quiz”

**Termin oddania:** 9 kwietnia 2026  
**Autor:** Hubert Strzelczyk

## Opis projektu
Aplikacja „Mini Quiz” to mobilne narzędzie edukacyjne stworzone w środowisku Android Studio. Pozwala użytkownikowi na rozwiązanie testu wielokrotnego wyboru, składającego się z 5 losowo wybranych pytań z puli dostępnej w kodzie aplikacji. Aplikacja monitoruje postęp użytkownika, nalicza punkty za poprawne odpowiedzi i wyświetla finalny komunikat po zakończeniu testu.

## Funkcjonalności
* Ekran startowy z przyciskiem rozpoczęcia quizu.
* Interfejs wyboru odpowiedzi (A, B, C).
* Dynamiczne losowanie 5 pytań z tablicy obiektów.
* Aktualizacja wyniku w czasie rzeczywistym.
* Funkcja resetowania quizu, pozwalająca na ponowne podejście.
* Powiadomienie (Toast) z wynikiem końcowym po ostatnim pytaniu.

## Instrukcja uruchomienia
1.  Pobierz repozytorium na dysk lokalny lub sklonuj je za pomocą komendy:
    `git clone https://github.com/HubertStrzelczyk/mini-quiz-strzelczyk.git`
2.  Otwórz program **Android Studio**.
3.  Wybierz opcję **Open** i wskaż folder z pobranym projektem.
4.  Poczekaj na zakończenie synchronizacji plików Gradle.
5.  Podłącz urządzenie z systemem Android z włączonym debugowaniem USB lub uruchom **Android Emulator** (Virtual Device).
6.  Kliknij ikonę **Run** (zielony trójkąt) w górnym pasku narzędziowym.
